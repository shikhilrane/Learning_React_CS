Components : 
    01. Components are also called as functions
    02. We can make function base Components as well class base Components
    03. Both type of components have hooks and can handle state 
    04. But mostly people use function based components.
    05. Components are function, that return you HTML type of code to render (HTML type of code called JSX)
    06. If we want to create navbar for the project then we will make seperate .jsx file for it and If we want to create footer for the project then we will make seperate .jsx file for it. If we want to create 2 side for the project then we will make seperate .jsx file for it and use that components for 2 times (it increases reusability).
    07. We write each and every components in seperate .jsx file
    08. We should make our components in seperate components folder which must be in src.
    09. To style a component we can create .css file with same name of component name.
    10. We should give className to style instead of class to assign class for element in .jsx file. Becasue class has another meaning in JS
    11. First letter of our component must be a capital letter
    12. Linking rules : 
        linking an image in element : 
            step 1 - Import an image (e.g. import Erling_Haaland_2023 from '../assets/Erling_Haaland_2023.jpg')
            step 2 - use image name in src of img tag in {}, (e.g. src={Erling_Haaland_2023})
            step 3 - to use css on component, make css file and import it in component (e.g. import "./UserCard.css")
        linking an component with App.jsx : 
            step 1 - import a custom component in App.jsx (e.g. import UserCard from "./components/userCard")
            step 2 - import App.css file in App.jsx if want to apply style (e.g. import "./App.css";)
    13. When using an component make sure that, component always returns a single tag or just use <></> (fragment). So always wrap things inside a single element or fragment
        const UserCard = () => {
            return (
                <div>       // single tag
                </div>
            )
                OR
            return (
                <>          // fragment
                </>
            )
        }
    14. We can use inline-css like style={{display:flex},{align-items:center}}




props : 
    1. props in JSX are as same as attribute in HTML element
    2. props can be use to pass different values in same component
    3. steps to use props 
        step 1 : pass a props in App.jsx with value
        step 2 : mention "props" as parameter in component
        step 3 : to inject those values in component use {prop.propName} (propName is a attribute we pass in App.jsx)
    4. we can even pass style as props as we did in code



JSX :
    1. JSX stands for JavaScript XML
    2. JSX is mixture of JS in HTML
