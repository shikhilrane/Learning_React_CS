# ⚡ Node.js + npm + nvm Cheatsheet

---

1️⃣ Node.js Commands

1.1 Check Node Version (Shows the installed Node.js version)
node -v

1.2 Run JavaScript File (Runs a JavaScript file using Node.js runtime)
node index.js

Example (Runs the app.js file)
node app.js

---

2️⃣ npm (Node Package Manager)

2.1 Check npm Version (Shows the installed npm version)
npm -v

2.2 Initialize Project (Creates package.json by asking project details)
npm init

Quick setup (Creates package.json with default values)
npm init -y

2.3 Install Package (Downloads and adds a package to project dependencies)
npm install package-name

Example (Installs Express framework)
npm install express

2.4 Install Dev Dependency (Installs package used only for development)
npm install package-name --save-dev

Example (Installs nodemon for development auto restart)
npm install nodemon --save-dev

2.5 Install Global Package (Installs a package globally on the system)
npm install -g package-name

Example (Installs nodemon globally)
npm install -g nodemon

2.6 Remove Package (Removes a package from project dependencies)
npm uninstall package-name

Example (Uninstalls Express from project)
npm uninstall express

2.7 List Installed Packages (Shows installed packages in the current project)
npm list

Global packages (Shows globally installed packages)
npm list -g

2.8 Update Package (Updates a specific package)
npm update package-name

Update all packages (Updates all dependencies)
npm update

2.9 Install All Dependencies (Installs all packages from package.json)
npm install

2.10 Check Outdated Packages (Shows packages with newer versions available)
npm outdated

2.11 Clean npm Cache (Clears npm cache to fix install issues)
npm cache clean --force

---

3️⃣ npx Commands

3.1 Run Package Without Installing (Runs a package without global installation)
npx package-name

Example (Creates React app without installing globally)
npx create-react-app my-app

---

# 4️⃣ nvm (Node Version Manager)

4.1 Check nvm Version (Shows installed NVM version)
nvm --version

4.2 Install Latest Node (Installs the latest Node.js version)
nvm install node

4.3 Install Specific Node Version (Installs a specific Node version)
nvm install 20

4.4 Install Latest LTS Version (Installs the latest long term support version)
nvm install --lts

4.5 Use Node Version (Switches terminal to a specific Node version)
nvm use 20

4.6 Use Latest Node (Switches to latest installed Node version)
nvm use node

4.7 Use Latest LTS (Switches to latest LTS version)
nvm use --lts

4.8 Use System Node (Uses system installed Node instead of NVM)
nvm use system

4.9 List Installed Versions (Shows Node versions installed with NVM)
nvm ls

4.10 List Available Versions (Shows all available Node versions)
nvm ls-remote

4.11 List Available LTS Versions (Shows available LTS versions)
nvm ls-remote --lts

4.12 Set Default Node Version (Sets default Node version)
nvm alias default 20

4.13 Show Aliases (Shows configured NVM aliases)
nvm alias

4.14 Uninstall Node Version (Removes a Node version from system)
nvm uninstall 18

---

# 5️⃣ Example Development Workflow

Install Node Version
nvm install 20

Use Node Version
nvm use 20

Create Project
npm init -y

Install Dependency
npm install express

Run Application
node index.js
