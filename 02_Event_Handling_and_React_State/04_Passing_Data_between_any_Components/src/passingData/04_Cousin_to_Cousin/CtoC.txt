Cousin → Cousin Data Passing (Communication)

There are 3 ways to pass data from cousin to cousin :
    1. Lift the state up to common ancestor
        1.1 The state is moved to the nearest common parent component.
        1.2 One cousin sends data to the parent, and the parent passes it to the other cousin through props.
        1.3 Flow : Cousin A → Parent → Cousin B
        1.4 Drawback : If many components are between them, the data must pass through each one. This is called Prop Drilling.

    2. Use React Context for global state management
        2.1 Context allows data to be stored in a shared place.
        2.2 Any component inside the Context can access the data without passing props manually.
        2.3 Advantage : Avoids prop drilling.

    3. Use state management library like Redux or Zustand
        3.1 The application state is stored in a global store.
        3.2 Any component can read or update the state from this store.
        3.3 Used when : The application becomes large and state management becomes complex.
