A. Installation of react : 
    There are 2 types of installation

    1. First way (create-react-app (it is depricated))
        - It is famous utility but problem is it is bulky utility so it gives big bundle and package size
        - npm(node package manager) - every language has their own package manager which allows us to install libraries like node(at local machine we called JS as node) has npm
        - Sometimes we don’t want to install a package globally or locally. We just want to run it once. In that case, we use npx (Node Package Executor), which executes the package directly without permanently installing it.
        - Installation 
            Step 1 - get in the folder in which we want to initiate our react project using commands like,
                        1. pwd - present working directory
                        2. ls - items of pwd
                        3. cd .. - to back from folder
                        4. cd folder_name - to move in the folder which is shown in ls
            Step 2 - write 'npx create-react-app 01basicreact'
                here, create-react-app = Utility (software) using this utility we can create project
                01basicreact = name of the file
        (it is time consuming process and take more time to initialise project)



    How to run project - 
        To run project we have to run start script which is listed in package.json
        Step 1 : first make sure that you are in the project using cd project_name (project name must be in small letters) command in terminal and use ls command to see if there is package.json present in the project
        step 2 : use 'npm run start' in terminal (same way we can use npm run build)
                 to exit from 'npm run start', Press 'Control+c'
        step 3 (use to push our app to production) : use 'npm run build' in terminal and build folder is created (our src folder's files get converted to vanilla js, css files in build folder(when we push our application in production then this build folder is serve to the user and not src)).







    2. Second way (Using vite, parcel (it is bundler, we write JS in multiple files and then it combine all those files to single file))
        - It is most useful way to create react projects
        - Installation 
            Step 1 - use 'npm create vite@latest' in terminal and press enter
            Step 2 - Give project name in all small letters and press enter
            Step 3 - select react and press enter
            Step 4 - select JavaScript
            (Now we dont have node_modules folder in it, so to install node_modules we should use following steps)
            Step 5 - use cd project_name to get in the folder & check using command 'ls' if package.json present in folder
            Step 6 - use 'npm install' in terminal to install node_modules
            Step 7 - to run use 'npm run dev' in terminal
                     use 'control + c' to escape






❓ Kya Vite = React?
🔴 Nahi, Vite React nahi hai.

✅ To Vite kya hai?
Vite ek build tool aur development server hai — jo React (ya kisi aur framework jaise Vue, Svelte, etc.) ke saath kaam karta hai.

🔄 React + Vite = Fast Setup + Fast Build

React → UI banane ke liye JavaScript library
Vite → React app ko fast start & build karne ke liye tool



B. Installation of tailwind in react (using Vite) :
    1. first make react project and be in the that folder
    2. run this command in terminal 'npm uninstall @vitejs/plugin-react' 
        because @vitejs/plugin-react@6 requires Vite 8, while the Tailwind Vite plugin currently supports Vite 5 / 6 / 7. (Kyuki @vitejs/plugin-react@6 Vite 8 require karta hai, aur Tailwind plugin Vite 7 tak support karta hai.)
    3. then run this command in terminal'npm install -D @vitejs/plugin-react@5' 
        because @vitejs/plugin-react@6 requires Vite 8, while the Tailwind Vite plugin currently supports Vite 5 / 6 / 7. (Kyuki @vitejs/plugin-react@6 Vite 8 require karta hai, aur Tailwind plugin Vite 7 tak support karta hai.)
    4. put 'npm install tailwindcss @tailwindcss/vite' in terminal to install tailwind in project
    5. Then import 'import tailwindcss from '@tailwindcss/vite'' in vite.config.js which is in root folder of project
    6. Then add put 'tailwindcss()' with plugins array. (plugins: [react(),tailwindcss(),] like this)
    7. Put '@import "tailwindcss";' in index.css

    NOTE : On 17/03/2026, step 2 and 3, Ye temporary compatibility workaround hai. Tumhe ye steps tab tak follow karne padenge jab tak Tailwind CSS ka Vite plugin Vite 8 support officially release nahi karta.
